Sistema De Control De Acceso CEET

Sistema de información para el registro y getión de los datos de ingreso y salida de usuarios en el centro Electricidad Electronica y Telecomunicaciones CEET

Ver Registros

Objetivos

Desarrollar e implementar un sistema de información enfocado al control de acceso de los administrativos, docentes y estudiantes, a la entidad educativa SENA sede (Electricidad, electrónica y telecomunicaciones).

Proposito

Promover un nuevo método de acceso, enfocado en el uso de tecnología (sistema de información), para mejorar la seguridad en la institución.

Sistema

Promover una alternativa tecnologica que este a la vanguardia, y que a su vez facilite el ingreso y salida desde un punto de vista optimo y seguro.

Funciones

Plantear un mecanismo efectivo donde se implemente el uso de un codigo de barras y un sistema web para la visualización de los registros de los usuarios.

Registros De Usuarios

Un sistema de acceso tiene como fin garantizar una seguridad y una gestión del ingreso de las pesonas de forma ordenada y sin complicaciones. Se entiende que un registro individual por persona es necesario, esto haciendo referencia a los datos de ingreso y salida pues componen la columna vertebral del principio de los sistemas de acceso, esto añadido a la fechas forman una estructura de orden y supervision, para la auditoria de el movimiento particular de las personas.

Preguntas frecuentes

En la siguiente sección se muestran algunas de las preguntas más frecuentes que realizan algunos usuarios.

¿Qué es el sistema de acceso CEET

Es una plataforma web donde las persona que se encuentren registradas podran visualizar sus registros de ingreso y salida, que efectuen de manera fisica en la institucion Centro de Electricidad, Electronica y Telecomunicaciones CEET, asi como tambien las fecha de asistencia en los cuales se presento

¿Para quien está dirigido?

Este aplicativo se encuentra disponible para cualquier persona que desee tener un registro de sus accesos y salidas de la institucion, por ejemplo: Aprendices, Instructores, Administrativos.

¿Cómo puedo ver mis registros?

Para poder ver los registros, primero debe realizar la inscripción en el aplicativo web presente, una vez realizado el procedimiento, se procede con el logueo en la plataforma, en la esquina superior derecha se podra visualizar un enlace llamado generar codigo, este le permitirá generar un codigo de barras que lo podra usar en momento de entrar a la institucion, cada acceso y salida que se haga con el codigo automanticamente quedara almacenado dentro de la base de datos. Cuando quiera ver sus registros, ingresa a el aplicativo y simplemente da en le boton "ver registros".
